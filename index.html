Anomaly Detection
- A Network Security Problem

**In this project, I delve into machine learning concepts with a specific focus on cybersecurity. Utilizing the UNSW-NB15 dataset, I address anomaly detection in network security by employing supervised learning techniques to model nine attack types and normal vectors. Through the utilization of algorithms such as Principal Component Analysis, Multiclass Support Vector Machines, Gradient Boosting Machines, and Logistic Regression, my objective is to enhance my knowledge in cybersecurity and trust and safety.
