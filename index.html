Markov Chains & PageRank
- An Airline Industry Problem

This project is a deep dive into a practical application of Markov Chains and the famous PageRank algorithm by Google within the domain of airline operations. By conceptualizing an airline's network of destinations as a graph, with airports serving as nodes and direct-flight connections as edges, the project aims to rank airports (nodes) in the network in terms of flight and passenger traffic patterns and revenue generated between each flight route using Markov chains, thereby unlocking hidden patterns not captured by descriptive statistics.

In the linked notebook on Github, I start off with exploratory data analysis (EDA) of an airline database with more than 5 tables using SQL and Pandas, focusing on computing various metrics such as the number of flights, passengers, and revenue between each of the different routes and preparing the data in a graph format necessary for applying PageRank algorithm , I then used SciPy to convert the data into a sparse matrix to optimise for efficiency in handling large datasets. As youâ€™ll see in greater detail in the notebook the whole idea of PageRank is about finding the most important airports in an airline network. To determine this importance, I assign weights to each airport based on factors like revenue, passenger numbers, and flight demand. Then, using techniques from linear algebra, probability theory, and tools like Numpy, I calculated the long-term probabilities associated with each rank. I also compared methods of ranking based on descriptive statistics (historical data) and that generated from the Markov Chain / PageRank algorithm.

Anomaly Detection
- A Network Security Problem

In this project, I delve into machine learning concepts with a specific focus on cybersecurity. Using the UNSW-NB15 dataset, I address anomaly detection in network security by employing supervised learning techniques to model nine attack types and normal vectors. I make use of algorithms such as Principal Component Analysis, Multiclass Support Vector Machines, Gradient Boosting Machines, and Logistic Regression to enhance my knowledge in cybersecurity and trust and safety.
